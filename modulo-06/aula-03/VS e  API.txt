CORS:

1. No: ferramentos / Gerenciador de Pacotes Nuget / Nuget Package Mangager Console: Install-Package Microsoft.AspNet.WebApi.Cors

2. Na classe `WebApiConfig.cs` adicionar a seguinte linha (em qualquer trecho do método Register): config.EnableCors(new EnableCorsAttribute("*", "*", "*"));
